# Portfolio Pessoal - Jeronimo Eduardo Fuck

Portfolio pessoal desenvolvido para apresentar minha transicao de carreira para desenvolvimento back-end/fullstack, reunindo projetos, experiencias profissionais, tecnologias, links sociais e formulario de contato.

## Deploy

🔗 https://jeronimoportfolio.vercel.app

## Sobre o projeto

O site foi criado como uma landing page interativa, com visual escuro inspirado em terminal/CLI, animacoes leves, cards expansivos de projetos e uma secao "Sobre mim" mais completa, destacando minha trajetoria da Engenharia Quimica para tecnologia.

## Funcionalidades

- Navegacao por secoes com header fixo.
- Hero com estilo terminal e background dinamico.
- Secao "Sobre mim" com texto biografico, fotos e status.
- Cards de projetos com visualizacao expandida.
- Galeria de imagens com lightbox.
- Projeto de game incorporado via iframe.
- Secao de tecnologias e ferramentas.
- Timeline de experiencias profissionais.
- Formulario de contato.
- Links para GitHub, LinkedIn e Instagram.

## Tecnologias
